# 利用梅尔倒频图谱的音乐流派分类
# 因数据集涉及到隐私,如需要请联系wanggang userbean@outlook.com
本项目聚焦音频领域的音乐流派分类任务，通过构建深度学习模型（基于 AudioCNN 架构），实现对不同音乐流派（如古典、摇滚、爵士、电子、嘻哈等）的自动识别。项目核心流程包括：将音乐信号转换为梅尔频谱等特征图谱，利用融合残差块、SE 注意力机制与 Mish 激活函数的 CNN 模型提取音频特征，最终通过全连接层完成流派分类。模型在公开音乐数据集（如 GTZAN、FMA）上表现稳定，可应用于音乐平台智能推荐、音乐库自动整理、流媒体内容标签生成等场景，为音频内容理解提供高效解决方案。

```
┌─────────────────────┐     ┌────────────────────────────────┐
│ 输入层              │     │ 音频特征图（13×H×W）            │
│ （如梅尔频谱）      │     │ 13=通道数，H/W=特征图高/宽      │
└────────────┬────────┘     └──────────────────┬─────────────┘
             │                                 │
┌────────────▼─────────────────────────────────▼─────────────┐
│ 初始卷积块：特征初步提取与通道扩张                          │
│ ┌─────────────┐  ┌──────────┐  ┌──────────┐                 │
│ │ Conv3×3     │→ │ BatchNorm2d │→ │ Mish激活  │              │
│ │ 13→64通道    │  │ （64通道）  │  │          │              │
│ └─────────────┘  └──────────┘  └──────────┘                 │
│ 输出尺寸：64×H×W（padding=1，stride=1，尺寸不变）            │
└────────────┬─────────────────────────────────┬─────────────┘
             │                                 │
┌────────────▼─────────────────────────────────▼─────────────┐
│ 残差块组1（layer1）：浅层特征细化                          │
│ ┌─────────────────────────────────────────────────────┐     │
│ │ 4个ResidualBlock（64→64通道，stride=1，groups=1）    │     │
│ │ 每个块含：Conv3×3 + BN + Mish + Conv3×3 + BN + SE模块 │     │
│ │ 残差连接：shortcut适配通道/步长，输出后再经Mish激活     │     │
│ └─────────────────────────────────────────────────────┘     │
│ 输出尺寸：64×H×W（stride=1，尺寸不变）                     │
└────────────┬─────────────────────────────────┬─────────────┘
             │                                 │
┌────────────▼─────────────────────────────────▼─────────────┐
│ 残差块组2（layer2）：特征下采样与通道扩张                  │
│ ┌─────────────────────────────────────────────────────┐     │
│ │ 4个ResidualBlock（64→128通道，首块stride=2，groups=2）│   │
│ │ 分组卷积（groups=2）：减少计算量，增强通道交互         │     │
│ │ 首块stride=2：特征图尺寸减半；后续块stride=1          │     │
│ └─────────────────────────────────────────────────────┘     │
│ 输出尺寸：128×(H/2)×(W/2)（下采样1次）                    │
└────────────┬─────────────────────────────────┬─────────────┘
             │                                 │
┌────────────▼─────────────────────────────────▼─────────────┐
│ 残差块组3（layer3）：深层特征提取                          │
│ ┌─────────────────────────────────────────────────────┐     │
│ │ 4个ResidualBlock（128→256通道，首块stride=2，groups=4）│  │
│ │ 分组卷积（groups=4）：进一步提升轻量化与通道多样性     │     │
│ │ 首块stride=2：特征图尺寸再减半；后续块stride=1        │     │
│ └─────────────────────────────────────────────────────┘     │
│ 输出尺寸：256×(H/4)×(W/4)（下采样2次，共减半2次）          │
└────────────┬─────────────────────────────────┬─────────────┘
             │                                 │
┌────────────▼─────────────────────────────────▼─────────────┐
│ 全局平均池化：特征压缩为向量                              │
│ ┌─────────────────────┐                                    │
│ │ AdaptiveAvgPool2d(1,1) │→ 将256×(H/4)×(W/4)压缩为256×1×1 │
│ └─────────────────────┘                                    │
│ 展平操作：从(Batch,256,1,1) → (Batch,256)                  │
└────────────┬─────────────────────────────────┬─────────────┘
             │                                 │
┌────────────▼─────────────────────────────────▼─────────────┐
│ 全连接层组：分类决策                                      │
│ ┌─────────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐   │
│ │ Linear(256→256)│→ │ BatchNorm1d │→ │ Mish激活  │→ │ Dropout(0.3)│ │
│ └─────────────┘  └──────────┘  └──────────┘  └──────────┘   │
│ ┌─────────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐   │
│ │ Linear(256→128)│→ │ BatchNorm1d │→ │ Mish激活  │→ │ Dropout(0.3)│ │
│ └─────────────┘  └──────────┘  └──────────┘  └──────────┘   │
│ ┌─────────────┐                                             │
│ │ Linear(128→num_classes) │→ 输出类别预测分数               │
│ │ num_classes=分类任务类别数（如10）                        │
│ └─────────────┘                                             │
└────────────┬─────────────────────────────────┬─────────────┘
             │                                 │
┌────────────▼─────────────────────────────────▼─────────────┐
│ 输出层                                                      │
│ 类别概率分布（经Softmax后）或原始预测分数                   │
└─────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────┐
│ Mish 激活函数公式：Mish(x) = x × tanh(softplus(x)) │
│ 计算流程：                                      │
│ 输入 x → softplus(x)（ln(1+e^x)） → tanh(softplus(x)) → x × 结果 → 输出 │
└─────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────┐
│ SE 模块（输入/输出：C×H×W，C=通道数）                     │
│                                                         │
│ ┌─────────────┐  ┌─────────────┐  ┌─────────────┐       │
│ │ 输入特征图   │→ │ 自适应平均池化 │→ │ 展平 (B×C)   │       │
│ │ (B×C×H×W)    │  │ → (B×C×1×1)  │  │             │       │
│ └─────────────┘  └─────────────┘  └──────┬──────┘       │
│                                          │              │
│ ┌─────────────┐  ┌─────────────┐  ┌──────▼──────┐       │
│ │ Linear(C→C/16)│← │ Mish激活    │← │ Linear(C/16→C)│     │
│ │ 通道压缩     │  │             │  │ 通道恢复     │       │
│ └──────┬──────┘  └─────────────┘  └──────┬──────┘       │
│        │                                 │              │
│        │  ┌─────────────┐  ┌─────────────┐              │
│        └─→│ Sigmoid     │→ │ reshape(B×C×1×1)│            │
│           │ 生成通道权重  │  │             │              │
│           └─────────────┘  └──────┬──────┘              │
│                                  │                       │
│ ┌─────────────┐  ┌──────────────▼─────────────┐          │
│ │ 输入特征图   │→ │ 逐通道相乘：输入 × 通道权重 │          │
│ │ (B×C×H×W)    │  │ （强化关键通道，抑制冗余） │          │
│ └─────────────┘  └──────────────┬─────────────┘          │
│                                 │                         │
│ ┌───────────────────────────────▼─────────────────────┐   │
│ │ 输出加权后特征图 (B×C×H×W)                          │   │
│ └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────┐
│ ResidualBlock（输入：in_channels×H×W；输出：out_channels×H'/W'） │
│                                                         │
│ 主分支（特征提取）：                                     │
│ ┌─────────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐ │
│ │ Conv3×3     │→ │ BatchNorm2d │→ │ Mish激活  │→ │ Conv3×3     │ │
│ │ in→out通道   │  │ （out通道）  │  │          │  │ out→out通道  │ │
│ │ stride=stride │  │            │  │          │  │ stride=1    │ │
│ └─────────────┘  └──────────┘  └──────────┘  └────┬─────┘ │
│                                                   │       │
│ ┌─────────────┐  ┌──────────┐                     │       │
│ │ BatchNorm2d │→ │ SE模块    │←────────────────────┘       │
│ │ （out通道）  │  │ （out通道） │                          │
│ └─────────────┘  └────┬─────┘                             │
│                       │                                    │
│ 残差分支（shortcut）：适配通道与步长差异                   │
│ ┌─────────────────────┐                                    │
│ │ 若in≠out或stride≠1：                                    │
│ │ Conv1×1（in→out，stride=stride） + BatchNorm2d          │
│ │ 若in=out且stride=1：直接恒等映射                        │
│ └─────────────────────┘                                    │
│        ┌───────────────────────────────────┐              │
│        │ 主分支输出 + 残差分支输出（元素相加） │              │
│        └────────────────────┬──────────────┘              │
│                             │                             │
│ ┌───────────────────────────▼────────────────────┐        │
│ │ Mish激活 → 输出残差块结果（out_channels×H'×W'） │        │
│ └────────────────────────────────────────────────┘        │
└─────────────────────────────────────────────────────────┘
```
